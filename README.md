# MarioArcade

Технологии: C#, Windows Forms

## Сборка
1. Откройте `MarioArcade.sln` в Visual Studio 2022
2. Восстановите NuGet-пакеты (если есть)
3. Запустите проект

## Структура
- `Objects/` — игровые сущности
- `Managers/` — система управления
- `Data/` — загрузка уровней
- `UI/` — интерфейс
- `sprites/` — графические ресурсы
